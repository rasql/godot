Math¶

Since Pythagoras, we know that \( a^2 + b^2 = c^2\).

(1)¶\[ \begin{align}\begin{aligned}(a + b)^2 = a^2 + 2ab + b^2\\\begin{split}(a + b)^2 &= (a + b)(a + b) \\ &= a^2 + 2ab + b^2\end{split}\end{aligned}\end{align} \]

The equation (1) is a quadratic equation.
